EasyBuilder HMI Water Treatment System Project

Project Description

This project presents a Human-Machine Interface (HMI) for a comprehensive water treatment system, developed primarily using EasyBuilder. The system is designed to provide clean water by filtering a supply for either direct end-user consumption or storage. The HMI is crucial for real-time monitoring, precise control, and effective interaction with the system's two main operational phases: normal filtration and automated backwash (filter cleaning). This application demonstrates robust data visualization, alarm management, user access control, and historical trending, enhancing operational efficiency and safety.

Key Features & Screens:

Welcome & System Overview (Animated)
Hand-Off-Auto (HOA) Device Control
Multi-Level User Access & Security
System Usage Monitoring (Hourmeter & Cycle Count)
Configuration & Alarm Setpoint Adjustment
Real-time Schematic Diagram Monitoring
Comprehensive Alarm Management & History
Process Value Trending & Analysis
