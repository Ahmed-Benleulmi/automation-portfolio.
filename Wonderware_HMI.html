Wonderware InduSoft Web Studio HMI Project

Project Description

This project showcases a multi-screen Human-Machine Interface (HMI) developed using Wonderware InduSoft Web Studio for a complete filtration plant. The HMI provides comprehensive control and monitoring of the filtration process, including the ability to start and stop the system, observe filter and tank levels, and manage the system's operational phases. The project emphasizes user security with different access levels, detailed control over system components, and robust alarm management. Real-time data visualization and historical trending provide valuable insights into system performance.

Key Features & Screens:

Main Screen (System Control & Overview)
Security Screen (User Access Management)
Control Screen (Manual/Auto Mode Selection)
Runtime Screen (Usage Monitoring & Reset)
Configuration Screen (Setpoint Adjustment)
Graph Screen (Schematic & Real-time Values)
Alarms Screen (Active Alarm Display & Reset)
Trending Screen (Historical Data Analysis)
