Connected Components Workbench

This upcoming project will delve into the capabilities of Rockwell Automation's Connected Components Workbench (CCW) software. We will develop a practical industrial control application, demonstrating programming across multiple paradigms. The project will feature implementations using Ladder Diagram (LD) for sequential control, Structured Text (ST) for complex algorithms and data manipulation, and Function Block Diagram (FBD) for visually representing control loops and logic. This will highlight CCW's versatility in handling diverse automation challenges.

ABB Automation Builder (CodeSys)

This exciting new project will explore industrial automation solutions using ABB Automation Builder, which incorporates the powerful CodeSys platform. We will implement a comprehensive control system demonstrating proficiency across all five IEC 61131-3 programming languages: Ladder Diagram (LD) for discrete logic, Structured Text (ST) for advanced data processing and calculations, Function Block Diagram (FBD) for intuitive visual programming, Instruction List (IL) for low-level control, and Sequential Function Chart (SFC) for structured process control. This project will showcase the robust capabilities of ABB's integrated development environment for diverse and complex automation applications.
