PLC Inventory Control (Barcode Scanner Integration)

Project Description

This project involves developing an inventory management system using RSLogix 500, integrating with a barcode scanner. The PLC will process strings from the scanner (e.g., "PPP-QQ:D" for Part Number-Quantity:Direction) to track on-hand quantities of three raw materials (widgets, doodads, and wockies). The program will differentiate between incoming (Direction 1) and outgoing (Direction 2) scans, updating inventory levels accordingly. It demonstrates data parsing, string manipulation, and maintaining accurate stock counts within a PLC environment, a common requirement in manufacturing and logistics.

Key Technologies & Features:

Allen-Bradley RSLogix 500
RSLinx Classic
RSLogix Emulate 500
Barcode Scanner Integration
String Manipulation
Inventory Management
Data Parsing
